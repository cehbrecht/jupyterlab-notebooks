# cchecker Demo


## Installation

Create conda environment:

    $ conda env create -f environment.yml
    $ source activate ccdemo

Install JupyterLab widget extension:

    $ jupyter labextension install @jupyter-widgets/jupyterlab-manager


## Start JupyterLab

    $ jupyter lab
